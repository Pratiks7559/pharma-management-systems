Keyboard Shortcuts Guide

Master these shortcuts to navigate your pharmacy management system efficiently

Navigation
  • Go to Dashboard
    Alt D
  • Navigate between inputs
    Esc
  • Focus search box
    Ctrl F
  • Print current page
    Ctrl P
Quick Actions
  • Delete selected item
    Ctrl D
  • Save form
    Ctrl S
  • Add new item
    Ctrl N
  • Edit current item
    Ctrl E
Dialog Controls
  • Confirm action
    Enter
  • Cancel/Close dialog
    Esc
  • Navigate between buttons
    Tab
  • Navigate backwards
    Shift Tab
Form Navigation
  • Next field
    Tab
  • Previous field
    Shift Tab
  • Submit form
    Ctrl Enter
  • Clear form
    Ctrl R

Pro Tips

  • The Ctrl+D shortcut works on any page with delete buttons
  • When multiple delete buttons are present, the system automatically selects the most relevant one
  • All confirmation dialogs auto-focus on the primary action button
  • Use Esc to navigate back through your input history
  • Keyboard shortcuts work even when buttons are not visible on screen
  • The system remembers your input values when navigating between fields
